# **Sales \& Profitability Analysis / Analyse des Ventes \& Profitabilité**



This project analyzes sales performance using an e-commerce dataset to evaluate profitability, discount impact, and regional performance. The goal is to uncover profit drivers, detect margin erosion, and provide actionable recommendations to optimize pricing and discount strategies.



Ce projet analyse la performance des ventes à partir d’un dataset e-commerce pour évaluer la rentabilité, l’impact des remises et la performance régionale. L’objectif est d’identifier les facteurs de profit, détecter l’érosion des marges et fournir des recommandations concrètes pour optimiser les prix et les stratégies de remise.



⚠️ Note / Remarque : The dataset does not include a time dimension, so temporal trends and seasonality analysis are not covered.

⚠️ Remarque : Le dataset ne contient pas de dimension temps, donc les tendances temporelles et la saisonnalité ne sont pas analysées.
